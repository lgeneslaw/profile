About Me

Projects


Tuberculosis Monitor:

For my senior capstone project at Tufts, my group of 3 students partnered with the University of Lima to build an app that tracks the health of TB patients by listening to how frequently they cough over time. Check it out on github, or check out this poster for more information.


NBA Python Projects:

The following python projects scrape the web (specifically basketball-reference.com) in order to store and manipulate NBA statistics/tables. Both projects are available on github.

Weighted Age of Championship Teams - Ranks the "age" of NBA championship teams.

NBA Finals Team Stats - Compiles the per-game statistics of every NBA finals team.


Task Manager:

Web app that displays views of upcoming and past tasks by day/week. User can create, update, and delete tasks through the web portal. Code available on github


Projects for Comp 150-Visualization:

While at Tufts, I took a programming course called Visualization that taught me the principles of bringing data to life. We used a software package called Processing to create java-based data visualizations and port them on the web.

Force-Directed Node-Link Graph - Simulates a self-structuring graph.

Theme River - Displays a flow of data over time.

Bouncing Circles - A simple project that handles the animation of bouncing circles.

Squarified Treemap - Displays a tree structure as rectangles within rectangles.
